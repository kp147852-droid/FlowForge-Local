# Project Brief: FlowForge Local

## Problem
Manual file workflows create operational drag and human error in teams handling recurring documents.

## Solution
FlowForge Local automates file intake and rule-based processing locally with auditability and reliability controls.

## Business value
- Reduces repetitive manual operations
- Improves consistency and traceability
- Creates structured logs for process analysis

## Technical highlights
- Rule engine with watcher + scheduler triggers
- Retry/backoff and quarantine behaviors
- FastAPI + SQLite backend with web UI
- Import/export templates for scalable adoption

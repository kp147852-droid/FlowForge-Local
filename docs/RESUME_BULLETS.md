# Resume Bullets (FlowForge Local)

- Built a local-first workflow automation platform using FastAPI, SQLite, and watchdog to automate file operations with rule-based triggers and scheduling.
- Designed reliability controls including dry-run previews, undo actions, retry/backoff logic, and quarantine-on-failure to reduce operational risk.
- Implemented observability via structured job logs, run history, and metrics endpoints to support process auditing and performance analysis.
- Added document and image processing actions (PDF text extraction, PDF merge, image conversion/compression) and integration hooks (webhook + CSV logging).
- Developed a browser-based rule builder with templates and JSON import/export to improve onboarding and workflow portability.

Built and shipped **FlowForge Local**, a local-first workflow automation system that watches folders, applies configurable file-processing rules, and logs each execution for auditability.
I designed the architecture end-to-end (API, rule engine, scheduler, and UI) and implemented production-style controls including retry/backoff, quarantine-on-failure, dry-run mode, and undo semantics.
The project demonstrates practical strengths for business analytics, data workflows, and applied AI tooling.
